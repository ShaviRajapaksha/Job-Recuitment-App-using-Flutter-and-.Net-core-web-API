# Job Recruitment API - Backend

## Project Overview
A complete .NET Core 8 Web API for a job recruitment platform with JWT authentication, PostgreSQL database, and role-based authorization. The application allows users to register as either Recruiters or Job Seekers. Recruiters can post job openings, while Job Seekers can browse available jobs and submit applications.

## Features
- ✅ JWT Authentication with token-based security
- ✅ Role-based Authorization (Recruiter/Job Seeker)
- ✅ PostgreSQL Database integration
- ✅ Entity Framework Core ORM
- ✅ Job Posting and Management system
- ✅ Job Application tracking
- ✅ Swagger/OpenAPI documentation
- ✅ Password Hashing with BCrypt
- ✅ CORS configuration for frontend access
- ✅ Input validation and error handling

## Technology Stack
- **Backend Framework**: .NET Core 8 Web API
- **Database**: PostgreSQL 15+
- **ORM**: Entity Framework Core 8
- **Authentication**: JWT (JSON Web Tokens)
- **Documentation**: Swashbuckle/Swagger
- **Security**: BCrypt.Net for password hashing
- **Package Manager**: NuGet
